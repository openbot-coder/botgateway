# botgateway 设计文档

## 项目概述

**项目名称:** botgateway  
**版本:** 0.1.0  
**描述:** AI Gateway for bots  
**项目类型:** Python 包项目

## 项目目标

提供一个AI网关，用于管理和路由各种机器人服务。

## 技术栈

- **编程语言:** Python 3.8+
- **包管理:** setuptools (pyproject.toml)
- **版本控制:** Git
